# PMB Online USNP

Sistem Penerimaan Mahasiswa Baru Online USNP 2024/2025

## Stack
- **Frontend**: React + Vite + Tailwind
- **Backend**: Node.js + Express + MySQL

## Setup Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

## Setup Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
